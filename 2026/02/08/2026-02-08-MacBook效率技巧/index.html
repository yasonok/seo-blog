MacBook 效率提升指南：高手都在用的 20 個技巧

MacBook 看似簡潔，但其實隱藏了許多強大功能。善用這些技巧，讓你的工作效率翻倍提升！

鍵盤快捷鍵大全

系統級快捷鍵

快捷鍵 功能
Command + C 複製
Command + V 貼上
Command + Z 復原
Command + Shift + Z 重做
Command + A 全選
Command + S 儲存

進階快捷鍵

  • Command + Tab：切換應用程式
  • Command + Option + Esc：強制結束應用程式
  • Control + Command + Space：叫出表情符號
  • Command + Shift + 4：截圖

Spotlight 搜尋技巧

按下 Command + Space 開啟 Spotlight，不只是搜尋檔案，還可以：

  • 進行快速計算
  • 匯率換算
  • 查詢詞典
  • 開啟應用程式

觸控板手勢

手勢 功能
雙指捏合 縮放
雙指左右滑動 切換頁面
三指上推 顯示所有視窗
四指張開 顯示桌面
從左邊緣右滑 喚醒螢幕保護程式

必裝效率 App

1. Alfred

比 Spotlight 更強大的搜尋啟動器，可以快速開啟檔案、應用程式，甚至執行計算。

2. Rectangle

免費的視窗管理工具，讓你用鍵盤快速調整視窗大小和位置。

3. Hammerspoon

強大的自動化工具，可以用 Lua 腳本自訂各種系統操作。

4. iTerm2

比內建 Terminal 更強大的終端機工具，支援分頁、分割視窗等功能。

常見問題

Q1：Mac 需要安裝防毒軟體嗎？

A： macOS 本身安全性很高，一般使用不需要額外防毒軟體。但仍需注意不要安裝來路不明的軟體。

Q2：如何讓 Mac 開機更快？

A： 減少登入時自動啟動的程式數量，並定期清理桌面和下載資料夾。

結論

善用 Mac 的內建功能和第三方工具，可以大幅提升工作效率。今天就從學習一個新技巧開始吧！

🚀 立即行動

  1. 背熟 5 個快捷鍵：今天就開始使用
  2. 下載 Alfred：體驗極速搜尋
  3. 整理桌面：減少視覺干擾
